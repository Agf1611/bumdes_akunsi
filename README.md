# Sistem Akuntansi BUMDes

Paket ini adalah source code bersih yang siap untuk **instalasi baru**.

## Isi paket
- aplikasi PHP/MySQL
- installer via browser (`/install.php`)
- template import Excel `.xlsx`
- dokumentasi pemasangan untuk:
  - XAMPP / localhost
  - shared hosting
- petunjuk penggunaan singkat

## Mulai cepat
1. Upload / extract source ke folder web.
2. Buat database MySQL kosong.
3. Akses `install.php` dari browser.
4. Isi data database dan buat akun admin.
5. Login, lalu lengkapi **Profil BUMDes**, **COA**, **Periode**, dan **Jurnal**.

Lihat folder `docs/` untuk panduan lengkap.
