# Petunjuk Penggunaan Singkat

## Urutan setup awal
1. **Profil BUMDes**
2. **Unit Usaha**
3. **COA**
4. **Periode Akuntansi**
5. **Jurnal Umum**
6. **Laporan**

## Import Excel
Template `.xlsx` tersedia di modul import umum:
- template COA
- template Jurnal

## Backup
Gunakan menu **Backup Database** sebelum melakukan perubahan besar.

## Tutup buku
- gunakan checklist periode terlebih dahulu
- pastikan jurnal seimbang
- pastikan laporan sudah benar
- baru lakukan tutup periode / tutup tahun

## Upload logo dan tanda tangan
Unggah dari menu **Profil**.
Format yang disarankan:
- JPG
- PNG
- WEBP
Ukuran maksimal: 2 MB
